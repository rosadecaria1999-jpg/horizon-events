# Horizon Events

## Author
Rosa Maria De Caria

## Project purpose
A mobile app for Elevate Horizon Connect, a local council that runs community events such as yoga, walking clubs, trivia nights, movie screenings and live music. The app helps residents find upcoming events and register for them on their phone. It is built to replace the spreadsheets and emails the council currently uses, so the staff have less manual work and residents have a faster, simpler way to take part in events.

## Core features
- View today's events on the home screen
- Browse the full list of events
- Search and filter events by keyword or date
- See event details
- Register for an event using a simple form
- Settings screen with text size, light/dark theme and sound on/off
- Bottom navigation bar (Home, Events, Register, Settings)

## Technology and tools identified
- React Native with Expo
- React Native Paper
- JavaScript
- fetch with async/await
- GitHub for version control
- Figma for wireframing
- VS Code as IDE
